##This project is an advanced NLP-based chatbot developed using Python, integrating libraries such as NLTK, spaCy, and Wikipedia. It processes user input through tokenization, lemmatization, and stopword removal to understand intent and generate meaningful responses. The chatbot includes features like Named Entity Recognition, sentiment analysis, context awareness, and conversation history tracking. It can respond dynamically to queries such as greetings, jokes, date/time requests, and general knowledge questions using Wikipedia integration. Additionally, it supports persistent self-learning by storing new responses provided by users, allowing it to improve over time. This project demonstrates practical applications of Natural Language Processing in building intelligent and interactive conversational systems.
